# Yelp Customer Sentiment Intelligence
**Advanced Text Mining & Natural Language Processing (NLP)**

## The Mission
To quantify the key drivers of restaurant success, I processed and analyzed an unstructured dataset of **48,000+ customer reviews**. The goal was to extract actionable insights and predict sentiment polarity to drive reputation management strategies.

## Technical Architecture
* **Data Pipeline:** Handled large-scale unstructured text using **NLTK** and **TF-IDF** for tokenization, stop-word removal, and vectorization.
* **Advanced Modeling:** Benchmarked multiple architectures, including **Logistic Regression**, **LSTM (Deep Learning)**, and **BERT Transformers** to classify review sentiment with high precision.
* **Sentiment Intelligence:** Utilized **VADER** and **TextBlob** to extract polarity scores from messy text data.

## Key Business Insights
* Discovered that mentions of "Food Quality" correlated **2x stronger** with 5-star ratings than "Service Speed."
* Delivered a robust sentiment classification system that translates raw customer feedback into actionable retention strategies.
